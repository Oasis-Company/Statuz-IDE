# 术语表

| 术语 | 英文 | 定义 |
|------|------|------|
| AuxiliaryBar | AuxiliaryBar | VS Code 右侧边栏，Statuz Sidebar 的挂载位置 |
| Board | Board | 策略画布，用于可视化展示和管理策略卡片（待实现） |
| CommandBar | CommandBar | 编辑器顶部命令栏，Statuz 扩展点之一 |
| Diff | Diff | 代码差异对比视图 |
| Fast Apply | Fast Apply | 快速应用 AI 生成的代码修改 |
| LLM | Large Language Model | 大语言模型（如 GPT、Claude） |
| MCP | Model Context Protocol | 模型上下文协议，AI 与外部工具的交互标准 |
| A2A | Agent-to-Agent | 智能体间通信协议（Statuz 自研，待实现） |
| Provider | Provider | LLM 提供商（如 OpenAI、Anthropic、Ollama） |
| Quick Edit | Quick Edit | 快速编辑功能（`Ctrl+K`） |
| scope-tailwind | scope-tailwind | 将 Tailwind 类名作用域化的构建工具 |
| tsup | tsup | 基于 esbuild 的 TypeScript 打包工具 |
| ViewContainer | ViewContainer | VS Code 视图容器，管理一组 ViewPane |
| ViewPane | ViewPane | VS Code 视图面板，Statuz Sidebar 的容器 |
| Widget | Widget | 编辑器内嵌组件（如 CommandBar） |
| Codicon | Codicon | VS Code 内置图标字体 |
| ESM | ECMAScript Modules | JavaScript 模块标准 |
| Onboarding | Onboarding | 首次使用引导流程 |
| Sandboxer | Sandboxer | 独立项目，策略规划应用，Statuz Board 的架构参考来源 |
