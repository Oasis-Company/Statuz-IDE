Core Idea

VS Code lets you edit code.
Statuz lets you understand it.

Three primitive queries that form the foundation of topological reasoning about software systems.

Traverse

What does this connect to? Explore the adjacency list of any node to discover its relationships.

traverse(node, relation?)
Impact

If this changes, who is affected? Reverse BFS blast radius analysis for change impact.

impact(node)
Path

How do I get from A to B? BFS shortest path discovery across the graph.

path(from, to)
